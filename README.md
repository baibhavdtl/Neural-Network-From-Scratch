# Neural-Network-From-Scratch
This is a small Neural Network trained on nnfs.io spiral dataset, implemented using NumPy and Python, without using any deep learning frameworks like TensorFlow, Keras or Pytorch.

# Features
1. Fully custom Neural Network Implementation
2. Forward and Backward Propagation from Scratch
3. ReLU activation function in hidden layers
4. Softmax activation function in the output layer
5. Cross-entropy loss for loss calculation

# Model Architecture
1. Input : Point on a 2D plane (x,y)
2. Hidden Layers: 64 Neurons (ReLU)
3. Output Layers: 3 Classes (Softmax)

# Dataset
This proect uses the NNFS.io Dataset:
300 training samples
3 classes
